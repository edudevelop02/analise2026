RESUMÃO

Principais Comandos Git
Configuração Inicial:
git config --global user.name "Seu Nome": Define nome.
git config --global user.email "email@exemplo.com": Define e-mail.

Iniciar
git init: Inicializa um repositório local na pasta atual.

Fluxo de Trabalho (Staging/Commit):
git status: Mostra arquivos modificados e staged.
git add <arquivo> ou git add .: Adiciona arquivos à área de stage.
git commit -m "mensagem": Grava o snapshot do stage no histórico.

Branches (Ramificações):
git branch: Lista branches.
git checkout -b <nome>: Cria e muda para um novo branch.
git checkout <nome>: Alterna para um branch existente.

Repositório Remoto (GitHub):
git remote add origin <url>: Conecta o local ao GitHub.
Mudando o repositorio:
exemplo 1: associando pasta a outro repositorio github 
git remote set-url origin https://github.com/edudevelop02/analise2026.git 
git remote -v       # Verifique o repositório atual


git push -u origin <branch>: Envia commits para o GitHub.
git pull origin <branch>: Recebe e mescla alterações do GitHub.
