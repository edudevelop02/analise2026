instalar o git


git init  -> inicializa a pasta do git
git add . (git add nome.py)  -> adiciona os arquivos para o git
git commit -m "primeiro commit"  - prepara os arq. para o git
git status -> verifica o status do commit
git log (histórico dos commits) - histórico do commit
